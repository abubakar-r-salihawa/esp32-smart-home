# Contributing

Contributions are welcome! Before contributing, please open an issue to discuss your ideas or report bugs.

## Process

1. Fork the repository and clone it locally.
2. Create a new branch for your feature or fix.
3. Follow the project's style guidelines (PEP 8 for Python) and write clear, concise commits.
4. Add tests or examples if appropriate.
5. Submit a pull request describing your changes and why they should be merged.

Thank you for helping improve this project!
